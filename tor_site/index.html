install nginx php7 mysql server in ubuntu

获取root权限，进入/root目录

sudo su
cd ~

更新软件源

apt-get update

添加php软件源，并且安装php7.3 mysql nginx

apt-add-repository ppa:ondrej/php
apt-get install mysql-server php7.3 php7.3-fpm php-mysql nginx tor -y

初始化数据库，并修改密码输入：

mysql

进入数据库，再输入：（将引号中的password改成你想设置的密码）

alter user root@localhost identified with mysql_native_password by 'password';

退出

quit

测试一下，输入：

mysql -uroot -p

会出现输入密码的提示，输入刚才的密码，回车，输入下面的命令测试一下：

show databases;

无误后

quit

退出

进入目录，修改php配置文件php.ini

cd /etc/php/7.3/fpm
vim php.ini

寻找取消下面语句的注释，去掉前面的分号

cgi.fix_pathinfo=1
extension=curl
extension=fileinfo
extension=gd2
extension=mbstring
extension=mysqli
extension=pdo_mysql

进入网站根目录，写一个php文件

cd /var/www/html
vim index.php

输入以下内容

<?php
phpinfo();
$sql = mysqli_connect('localhost', 'root', 'password');
if(!$sql) echo "fail\n"; else echo "ok\n";
?>

运行一下该文件（输入下面的命令），会出现phpinfo以及最后一行的ok，证明php成功连接mysql数据库，否则有错误

php -f index.php

进入目录编辑nginx配置文件，让nginx生成网页时可以运行php代码

cd /etc/nginx/sites-available
vim default

找到location ~ \.php$ 先取消掉注释把{}里的内容改成：

listen 8080 default_server; location ~ \.php$ {
try_files $uri =404;
fastcgi_split_path_info ^(.+\.php)(/.+)$;
fastcgi_pass unix:/run/php/php7.3-fpm.sock;
fastcgi_index index.php;
fastcgi_param SCRIPT_FILENAME $document_root$fastcgi_script_name;
include fastcgi_params;
}

找到location ~ /\.ht 取消注释把{}里的内容改成：

location ~ /\.ht {
deny all;
}

配置tor文件

cd /etc/tor
vim torrc

找到相关代码，取消注释，修改成：

HiddenServiceDir /var/lib/tor/other_hidden_service/
HiddenServicePort 80 127.0.0.1:8080

启动服务

cd ~
service php7.3-fpm start
service nginx start
service nginx restart
service tor start
service tor restart

域名：

cd /var/lib/tor/hidden_service
cat hostname
