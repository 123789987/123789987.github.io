Shadowsocksr搭建使用教程

服务器端，以ubuntu 为例

进入root账号

sudo su cd ~

更新软件源

apt-get update

安装python

apt-get install python-pip

安装git

apt-get install git

下载Shadowsocksr，这里注意不要安装来源不明的软件，有的有后门，可能会被做成肉鸡

git clone -b manyuser https://github.com/shadowsocksr-backup/shadowsocksr.git

初始化

cd ~/shadowsocksr
bash initcfg.sh

此时生成一个user-config.json文件，编辑这个配置文件

vim user-config.json

输入i进入编辑模式
其中，
server_port是服务端对客户端开的端口，默认是8388
password是密码，默认是m
method对应客户端的加密
protocol对应客户端的协议
protocol_param对应协议参数
obfs对应客户端的混淆，建议是plain或者是http_simple，被墙的几率小些
obfs_param对应混淆参数

如果想让一个服务器的多个端口针对不同用户服务，可以这么修改文件：
先去掉server_port和password两行
再在password那一行输入：（字符不要错，逗号不能多，不能少） "port_password" : {
"8388" : "m",
"8389" : "m"
},

编辑完后保存退出，接着启动服务

cd shadowsocks
python server.py -d start

停止服务
python server.py -d stop
重启服务python server.py -d restart
