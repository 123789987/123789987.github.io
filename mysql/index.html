mysql在Windows安装

Download

https://dev.mysql.com/get/Downloads/MySQL-8.0/mysql-8.0.17-winx64.zip

初始化

cd到相应根目录下的bin目录，或者将bin目录加入环境变量 # 会输出密码 mysqld --initialize --console # 可以没有密码登录 mysqld --initialize-insecure

安装

mysld -install

启动

net start mysql

停止进程

net stop mysql

登入

# 如果可以没有密码登入 mysql -uroot --skip-password # 否则 mysql -uroot -p # 接着根据提示输入密码

修改密码

alter user root@localhost identified with mysql_native_password by 'password';

打印数据库名称

show databases;

退出

quit; exit;
